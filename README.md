# ❤️ Love Score Calculator

A fun **Love Score Calculator** built with **HTML, CSS, and JavaScript**.
Enter two names and the app will generate a random love compatibility score.

## 📌 Features

* Enter **your name** and **your crush's name**
* Generates a **random love percentage**
* Displays a **custom message based on the score**
* **Reset button** to try again
* Responsive design for smaller screens

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Google Fonts

## 🚀 How It Works

1. Type your name in the first input field.
2. Type your crush's name in the second input field.
3. Click the **Calculate** button.
4. The app generates a **random love score (1–100%)** and displays a message.
